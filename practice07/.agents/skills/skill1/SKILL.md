## 文件分析技能

本技能用于分析指定文件的内容，包括：

### 功能
1. 读取文件内容
2. 统计文件行数、字符数、单词数
3. 识别文件类型
4. 提取关键信息

### 使用方法
调用此技能时，需要提供文件路径作为参数。

### 输出格式
分析结果将以结构化的 JSON 格式返回，包含以下字段：
- file_path: 文件路径
- line_count: 行数
- character_count: 字符数
- word_count: 单词数
- file_type: 文件类型
- content_preview: 内容预览（前100个字符）
