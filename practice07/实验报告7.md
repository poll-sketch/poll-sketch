# 实验报告7：AI智能体链式工具调用

## 实验目的

本实验旨在基于practice06的技能管理系统，实现链式工具调用（Chained Tool Calls）功能，让AI智能体能够根据中间结果自主决定下一步工具调用。具体目标包括：

1. **开发链式调用上下文管理器**
   - 创建一个类用于在多个工具调用之间传递数据和状态
   - 记录每一步的调用和结果
   - 存储中间变量供后续步骤使用
   - 设置最大迭代次数，防止无限循环

2. **开发链式调用执行函数**
   - 实现链式工具调用的完整流程
   - 初始化消息历史，包含system prompt
   - 循环最多max_iterations次进行决策
   - 构建分析提示词，调用LLM决定下一步操作
   - 解析LLM响应，支持JSON格式
   - 如果任务完成，返回最终回答
   - 如果需继续调用，执行工具并记录到上下文

3. **开发分析提示词构建函数**
   - 提示词包含用户原始请求
   - 提示词包含已执行的工具调用历史
   - 提示词包含决策规则说明
   - 提示词包含JSON输出格式要求

4. **实现LLM输出格式规范**
   - 完成任务时：`{"done": true, "answer": "最终回答内容"}`
   - 继续调用工具时：`{"done": false, "tool_call": {"name": "工具名称", "arguments": {...}}}`

5. **更新系统提示词**
   - 说明工具调用的顺序依赖关系
   - 指导LLM如何根据中间结果决定后续操作
   - 提供链式调用的示例
   - 说明上下文变量的使用方式

### 2. 实现链式调用上下文管理器

**类设计：ChainedCallContext**

该类用于在多个工具调用之间传递数据和状态，主要功能包括：

**初始化方法：**
```python
def __init__(self, max_iterations=10):
    self.max_iterations = max_iterations
    self.current_iteration = 0
    self.call_history = []  # 记录每一步的调用和结果
    self.variables = {}    # 存储中间变量供后续步骤使用
    self.final_answer = None  # 最终回答
```

**核心方法：**

1. **add_call(tool_name, arguments, result)**
   - 添加工具调用记录
   - 包含迭代次数、工具名称、参数、结果和时间戳

2. **set_variable(name, value)**
   - 设置中间变量
   - 用于存储前一步工具的输出供后续步骤使用

3. **get_variable(name, default=None)**
   - 获取中间变量
   - 支持默认值

4. **increment_iteration()**
   - 增加迭代次数
   - 用于跟踪执行进度

5. **is_max_iterations_reached()**
   - 检查是否达到最大迭代次数
   - 防止无限循环

6. **get_summary()**
   - 获取链式调用的摘要信息
   - 包含调用历史、中间变量和最终回答

### 4. 实现链式调用执行函数

**函数设计：execute_chained_tool_call(env_vars, user_request, max_iterations=10)**

该函数实现了链式工具调用的完整流程。

**执行流程：**

**步骤1：初始化**
- 创建ChainedCallContext对象
- 设置最大迭代次数
- 打印开始信息

**步骤2：循环执行**
```
while not context.is_max_iterations_reached():
    # 2.1 构建分析提示词
    prompt = build_analysis_prompt(user_request, context)
    
    # 2.2 调用LLM决策
    response = call_llm_for_chain(env_vars, prompt)
    
    # 2.3 解析LLM响应
    decision = json.loads(response)
    
    # 2.4 判断任务状态
    if decision.get('done'):
        # 任务完成
        context.final_answer = decision.get('answer', '')
        break
    else:
        # 需要继续调用工具
        tool_call = decision.get('tool_call', {})
        tool_name = tool_call.get('name')
        arguments = tool_call.get('arguments', {})
        
        # 2.5 执行工具
        result = execute_tool(tool_name, arguments)
        
        # 2.6 记录到上下文
        context.add_call(tool_name, arguments, result)
        
        # 2.7 设置中间变量
        if result.get('status') == 'success':
            data = result.get('data')
            if data:
                var_name = f"{tool_name}_result"
                context.set_variable(var_name, data)
        
        # 2.8 增加迭代次数
        context.increment_iteration()
```

**步骤3：检查终止条件**
- 检查是否因为达到最大迭代次数而终止
- 如果未获得最终回答，提供错误信息

**步骤4：返回结果**
- 返回ChainedCallContext对象
- 包含完整的调用历史和最终回答
