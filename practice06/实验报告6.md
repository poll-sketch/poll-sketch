### 2. 实现 list_available_skills 函数

该函数的主要功能包括：
- 读取项目根目录下 `.agents/skills` 目录
- 遍历所有一级子目录
- 读取每个子目录中的 `SKILL.md` 文件
- 提取 YAML front matter（`---` 标记之间的内容）
- 解析 `name` 和 `description` 字段
- 返回技能列表

关键实现代码：
```python
def list_available_skills():
    skills = []
    project_root = os.path.dirname(os.path.dirname(__file__))
    skills_dir = os.path.join(project_root, '.agents', 'skills')
    
    if os.path.exists(skills_dir) and os.path.isdir(skills_dir):
        for skill_name in os.listdir(skills_dir):
            skill_dir = os.path.join(skills_dir, skill_name)
            if os.path.isdir(skill_dir):
                skill_file = os.path.join(skill_dir, 'SKILL.md')
                if os.path.exists(skill_file) and os.path.isfile(skill_file):
                    with open(skill_file, 'r', encoding='utf-8') as f:
                        content = f.read()
                    
                    if content.startswith('---'):
                        front_matter_end = content.find('---', 3)
                        if front_matter_end != -1:
                            front_matter = content[3:front_matter_end].strip()
                            name = ""
                            description = ""
                            for line in front_matter.split('\n'):
                                line = line.strip()
                                if line.startswith('name:'):
                                    name = line[5:].strip().strip('"')
                                elif line.startswith('description:'):
                                    description = line[12:].strip().strip('"')
                            if name:
                                skills.append({
                                    "name": name,
                                    "description": description
                                })
    
    return {"status": "success", "data": skills}
```

### 3. 实现 load_skill_content 函数

该函数的主要功能包括：
- 接收技能名称作为参数
- 定位对应的 `SKILL.md` 文件
- 读取文件内容
- 提取 YAML front matter 之后的部分作为技能正文
- 返回技能正文内容

关键实现代码：
```python
def load_skill_content(skill_name):
    project_root = os.path.dirname(os.path.dirname(__file__))
    skills_dir = os.path.join(project_root, '.agents', 'skills')
    skill_dir = os.path.join(skills_dir, skill_name)
    skill_file = os.path.join(skill_dir, 'SKILL.md')
    
    if not os.path.exists(skill_file) or not os.path.isfile(skill_file):
        return {"status": "error", "message": f"技能文件不存在: {skill_file}"}
    
    with open(skill_file, 'r', encoding='utf-8') as f:
        content = f.read()
    
    if content.startswith('---'):
        front_matter_end = content.find('---', 3)
        if front_matter_end != -1:
            content = content[front_matter_end + 3:].strip()
    
    return {"status": "success", "data": content}
```
