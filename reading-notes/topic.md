# 主题锚定规范 (Topic Anchoring)

## 1.1 核心主题声明

- **主命题**：《边城》中湘西世界呈现的"自然人性美"与"命运悲剧性"的交织，对当代青年在城市化生存语境下重新审视"何为美好生活"的精神启示
- **边界范围**：
  - **必须涵盖**：
    1. 《边城》的文学地位与沈从文创作背景
    2. 翠翠、傩送、天保三人的情感纠葛与悲剧命运
    3. 湘西自然人文景观的诗意呈现
    4. "茶峒"小城折射的传统伦理秩序
    5. 从软件工程学生视角反思"纯朴人性"的时代价值
  - **明确排除**：
    1. 不详细讨论沈从文生平年表及创作风格演变
    2. 不比较《边城》与其他乡土文学作品的差异
    3. 不分析该书的市场反响、出版销量或影视改编

- **目标读者**：高校文科/理工科通识课程助教、同学，以及对现代文学感兴趣的非专业读者

## 1.2 偏题检测机制

生成过程中，每完成一个章节需自检：

- [ ] 本节内容是否直接支撑"自然人性美与命运悲剧性的交织"这一主命题？
- [ ] 若删除本节，主命题是否仍然完整？（如答案是"是"，则本节可能冗余）
- [ ] 本节是否引入了边界范围外的新话题？

## 1.3 题材适配示例

| 题材类型 | 主命题示例 | 边界排除示例 |
|---------|-----------|------------|
| 软件工程报告 | 本系统如何通过微服务架构解决高并发场景下的性能瓶颈 | 不讨论前端UI设计细节、不对比其他公司商业产品 |
| **读书心得（本文）** | 《边城》中"自然人性"与"命运悲剧"的交织对当代青年精神困惑的启示 | **不介绍作者生平年表、不讨论该书出版市场反响** |

## 1.4 纠偏触发词

当内容偏离时，使用以下句式强制回归：

> "回到核心命题：《边城》中自然人性与命运的交织如何启示我们...，我们需要聚焦于..."
